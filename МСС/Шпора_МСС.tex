\subsection{Свойства сплошных сред}
\begin{definition} 
Сплошность - непрерывное распределение масс и физико-химических свойств в среде.
\end{definition}
\begin{definition} 
Внутренняя подвижность - деформация.
\end{definition}
\subsection{Уравнение движения несжимаемой среды}
Существует две модели, описывающих жидкости:
\begin{itemize}
\item Модель идеальной жидкости.
\item Модель вязкой жидкости.
\end{itemize}

\subsection{Интегральные законы сохранения массы, импульса и момента импульса}
\[ \rho = \lim_{\omega \rightarrow 0} \dfrac{M}{\omega}, \ \omega(t) \in \mathbb{R}^3\]
\[ M = \int\limits_{\omega(t)} \rho d\omega \]
\begin{definition} 
Жидкий объем $(\omega_t)$ - объем, состоящий из одних и тех же жидких частиц.
\end{definition}
Существуют два подхода:
\begin{itemize}
\item Эйлеров - система координат неподвижна,фиксируются изменения в каждой точке пространства для получения полной картины.
\item Лагранжев - система координат прикрепляется к материальной точке, измеряется её состояние в разные моменты времени, для полноты измеряются все материальные точки.
\end{itemize}
\begin{enumerate}
    \item $\Omega$ - конечно.
    \item $\omega_i$ - равновозможны (симметричны).
    \item $\forall A \subseteq \Omega, \  P(A) = \dfrac{\#(A)}{\#(\Omega)}$
\end{enumerate}
\subsection{Модель дискретной вероятности}
\begin{enumerate}
    \item $\Omega$ - не более, чем счетно.
    \item $\sum\limits_{i,  \omega_i \in \Omega} p_i = 1$
    \item $P(A) = \sum \limits_{i, \omega_i \in A} p_i$
\end{enumerate}
\subsection{Модель геометрической вероятности}
\begin{enumerate}
    \item $\Omega$ - ограниченная область в $\mathbb{R}^n$.
    \item $P(a) = \dfrac{\Lambda_n(A)}{\Lambda_n (\Omega)} $
\end{enumerate}
